
# Clinical Analysis of Diabetes Risk Factors 

## Project Overview
This project analyzes clinical markers from the Pima Indians Diabetes dataset to identify 
correlations between patient age, BMI, and glucose levels.

## Key Findings
* **Strongest Predictor:** Glucose shows a strong positive correlation (0.47) with diabetes outcomes.
* **Age Trends:** Glucose levels tend to rise more aggressively in diabetic patients as they age compared to the healthy control group.

## Tools Used
* **Python** (Pandas, Seaborn, Matplotlib)
* **Jupyter Notebook**
